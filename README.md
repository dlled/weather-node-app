# WEATHER - APP

A Node App to get the weather of a specific city.

## Installation

First check you have installed [nodejs](https://nodejs.org/es/) and [npm](https://www.npmjs.com/get-npm)

Download the project and install the dependencies. 
```bash
npm install
```

## Start

```bash
npm start
```
